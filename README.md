## Installatie 
Om te beginnen, zie je hieronder wat de stappen zijn om het programma te openen & te gebruiken!

1. Open Xcode & clone het programma.
```bash
Git Clone: https://github.com/a-racaj/To-Do-Lijstje.git
```
2. Run het programma.
```bash
Command R
```
3. Welkom!
